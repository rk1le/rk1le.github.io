How I Built This Website

I used GitHub to create a new repository for my website files. A repository is like a folder that stores my project online. Inside that folder, I added this HTML file and saved it using a commit, which records changes I make to the project. I worked on the main branch, which is the main version of my website. Once I committed my changes and pushed them to GitHub, I turned on GitHub Pages to make my site live on the internet. Each time I update and commit changes, my website automatically refreshes online.

My Learning Experience

This assignment helped me understand the basics of how websites are created and shared. I learned how GitHub can be used not just for code, but also for hosting websites. It was exciting to see my work appear on a real web link that anyone can visit!
